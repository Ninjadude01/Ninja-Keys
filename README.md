#Ninja-Keys

My project is two custom keycaps, with a ninja theme. One keycap features 3 small kunai knifes on the top, and another has shuikens. They are designed to fit on cherry mx switches. I decided to create this project because designing my very own keycaps sounded fun to me, and because many keyboard switches are plain and boring so why not create your own?
